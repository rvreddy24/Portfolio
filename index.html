Basic Environmental Monitoring System

Developed a real-time temperature and humidity monitoring system using ESP32 and DHT11, with LED indicators and a customizable user interface through a serial monitor.

Description: Monitors temperature and humidity in real-time and uses an LED to indicate levels based on custom thresholds. Users interact with the system to adjust thresholds and control monitoring status.

Components: ESP32 Microcontroller, DHT11 Sensor, Jumper Wires.

System Features:

    Temperature and Humidity Monitoring: Reads data every 2 seconds and displays it on a serial monitor.

    LED Indicator: Visual LED indication for levels above or below set thresholds.

    User Interface: Allows start/stop commands and threshold adjustments via a serial monitor.

Technologies Used: ESP32, DHT11 Sensor, Serial Communication, C++.

Outcome: Real-time environment monitoring system with user-defined thresholds for interactive control and customizable alerts.
